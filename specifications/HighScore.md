# Class HighScore
*Gather and save scores to file.*

## _Fields:_
- **name**
    - *type*: `String`
    - Holds name of player.

- **shotsCount**
    - *type*: `Integer`
    - Holds number of shots hitted by player.

- **date**
    - *type*: `Date`
    - Holds current date.

- **HighScore**

- **incrementShotCount**
    - *arguments*: `null`
    - *return*: `void`
    - Increment shotcount in instance.

- **toString**
    - *arguments*: `null`

- **writeToFile**
    - *arguments*: `null`
