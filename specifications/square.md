# Class Square
Models the smallest unit in Ships and board.

## _Fields:_
- **y**
    - *type*: `Integer`
    - Holds coordinates in board.

- **sign**
    - *type*: `String`
    - Holds sign that can be read and changed by Square methods.

## _Instance methods:_
- **Square**
    - *arguments*: `Integer, Integer`
    - *return*: `void`
    - Construct object class Square.

- **setSign**
    - *arguments*: `String`
    - *return*: `void`
    - Change field sign in instance for the one given as arguent.

- **getSign**
    - *arguments*: `null`
    - *return*: `String`
    - Return sign of instance.
